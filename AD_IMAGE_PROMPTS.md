## Color Specifications

When generating, specify these exact colors:
- **Background**: Pure black `#050505` or `#0a0a0a`
- **Accent**: Electric lime `#c3f53c`
- **Text**: Pure white `#ffffff`
- **Secondary**: Gray `#9ca3af`
- **Error/Problem**: Muted red `#ef4444` (dimmed)
- **Success**: Lime green `#c3f53c`

---

## Style Keywords to Include

Add these keywords to any prompt for consistency:
```
minimalist, dark mode, tech startup aesthetic, corporate professional,
high contrast, clean typography, geometric shapes, no photographs,
no illustrations of people, modern design, premium feel, 1:1 ratio
```

## Backup: Canva Alternative

If Nano Banana results aren't satisfactory, use Canva with these specs:

3. **Primary font**: Syne ExtraBold (or similar geometric bold)

5. **Elements**: Use abstract shapes, no stock photos

## Final Tips

1. **Test multiple versions** - Generate 2-3 variations of each prompt
2. **A/B test colors** - Try one ad with pure lime, one with green gradient
3. **Keep text minimal** - Max 7 words per ad image
4. **Mobile-first** - Text should be readable on small screens
5. **Add urgency** - Consider adding "Limited spots" or "This week only"
