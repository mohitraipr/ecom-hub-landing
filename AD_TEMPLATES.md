## Testing Strategy

### Day 1-2: Test Phase
1. Run all 3 ads with ₹300 each (₹900 total)
2. Set up as A/B test in Ads Manager
3. Let them run for 48 hours minimum

### Day 2 Analysis: Check these metrics
- **CTR (Click-Through Rate):** Should be >1%
- **CPM (Cost Per 1000 Impressions):** Should be <₹200
- **Cost Per Message:** Track which ad gets cheapest leads

### Day 3: Scale Winner
1. Pause the 2 worst-performing ads
2. Put remaining ₹1,800 on the winning ad
3. Monitor throughout the day

---

## Red Flags & Fixes

| Issue | What to do |
|-------|------------|
| CTR below 1% | Change the creative/image |
| CPM above ₹200 | Narrow targeting OR broaden to Advantage+ |
| No messages after ₹500 | Pause everything, fix landing page first |
| Lots of clicks, no messages | WhatsApp link might be broken - test it |

## Image Creation (Canva)

### Template 1: Authority Ad
1. Open Canva → Create Design → Instagram Post (1080x1080)
2. Background: Dark (#0a0a0a)
3. Split vertically
4. Left side: Add screenshots of cluttered dashboards, add red overlay
5. Right side: Add clean dashboard mockup, add green checkmarks
6. Add text: "We Manage 2 Lakh Orders/Month" in white, bold
7. Add ecom-hub logo in corner

### Template 2: Problem-Solution Ad
1. Same size, dark background
2. Use "Sad face" → "Happy face" concept
3. List problems with ❌ on left
4. List solutions with ✓ on right
5. Bold headline at top

### Template 3: Speed Ad
